## Security Architecture

```
┌──────────────────────────────────────────────────────────────────────────────────────┐
│                              SECURITY LAYERS                                          │
└──────────────────────────────────────────────────────────────────────────────────────┘

                                 INTERNET
                                    │
                                    ▼
┌─────────────────────────────────────────────────────────────────────────────────────┐
│  LAYER 1: CLOUDFLARE PROTECTION                                                      │
│  ├── 🛡️ DDoS Mitigation ────────── Absorbs volumetric attacks                       │
│  ├── 🔥 WAF Rules ───────────────── Blocks OWASP Top 10 attacks                     │
│  ├── 🤖 Bot Management ──────────── Filters malicious bots                          │
│  └── 🔒 SSL/TLS Encryption ──────── End-to-end HTTPS                                │
└─────────────────────────────────────────────────────────────────────────────────────┘
                                    │
                                    ▼
┌─────────────────────────────────────────────────────────────────────────────────────┐
│  LAYER 2: ZERO TRUST TUNNEL                                                          │
│  ├── 🚫 No Inbound Ports ────────── Server firewall blocks all inbound              │
│  ├── 🔐 Outbound Only ───────────── Tunnel initiates connection outward             │
│  └── 🎟️ Token Authentication ────── Secure tunnel authentication                    │
└─────────────────────────────────────────────────────────────────────────────────────┘
                                    │
                                    ▼
┌─────────────────────────────────────────────────────────────────────────────────────┐
│  LAYER 3: CONTAINER ISOLATION                                                        │
│  ├── 🐳 Docker Network ──────────── Isolated network namespace                      │
│  ├── 👤 Non-root User ───────────── Application runs as unprivileged user           │
│  └── 📦 Minimal Image ───────────── Alpine-based, reduced attack surface            │
└─────────────────────────────────────────────────────────────────────────────────────┘
                                    │
                                    ▼
┌─────────────────────────────────────────────────────────────────────────────────────┐
│  LAYER 4: APPLICATION SECURITY                                                       │
│  ├── ✅ Input Validation ────────── All form inputs validated                       │
│  ├── 🔑 Environment Variables ───── Secrets not in code                             │
│  └── 📝 Security Headers ────────── CSP, HSTS, X-Frame-Options                      │
└─────────────────────────────────────────────────────────────────────────────────────┘
```
